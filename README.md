# Personal Portfolio Website

This repository contains a GitHub Pages-ready personal portfolio for a data scientist and researcher.

## Project Structure

```text
.
|-- index.html
|-- about.html
|-- projects.html
|-- research.html
|-- skills.html
|-- resume.html
|-- contact.html
|-- 404.html
`-- assets
    |-- css/styles.css
    |-- js/main.js
    |-- images/
    `-- docs/mirwais-farahi-cv.pdf
```

## Where To Edit Personal Info

- Update your name, title, biography, email, LinkedIn, GitHub, and location across the HTML pages.
- Search for comments like `<!-- Replace with actual biography -->` and placeholders such as `your.email@example.com`.

## Where To Edit Projects

- Open `projects.html`.
- Replace each placeholder project block with your real project details, links, methods, and outcomes.

## Where To Add Publications

- Open `research.html`.
- Update the publications, working papers, reports, and contribution sections with your actual citation details.

## How To Replace The CV File

- Replace `assets/docs/mirwais-farahi-cv.pdf` with your real CV.
- Keep the filename the same to preserve all existing download links, or update the HTML links if you rename it.

## GitHub Pages Deployment

1. Push this repository to GitHub.
2. In repository settings, open **Pages**.
3. Deploy from the main branch root.
4. Save and wait for GitHub Pages to publish the site.

## Notes

- Shared styles live in `assets/css/styles.css`.
- Shared interactions live in `assets/js/main.js`.
- The contact form is currently a UI placeholder and needs a form backend for real submissions.
